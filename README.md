# kenote-react-startkit
This is a React development scaffolding

## Installation

```
git clone https://github.com/thondery/kenote-react-startkit.git
cd kenote-react-startkit && yarn install
```

## Usage

dev
```
yarn dev
```

compile
```
yarn compile
```
